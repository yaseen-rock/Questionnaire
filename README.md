## Getting Started

### Prerequisites

Before you begin, make sure you have the following installed:

- [Node.js](https://nodejs.org/) 
- [MongoDB](https://www.mongodb.com/) 

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/your-username/questionnaire-backend.git
   cd questionnaire-backend

2. Installation

```bash
$ npm install
```

3. Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

4. Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
